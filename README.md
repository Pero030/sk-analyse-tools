## 🛠️ GitHub Pages Einrichtung

1. **Repository erstellen:**
   - Gehe zu [github.com/new](https://github.com/new)
   - Repository-Name: `analyse-tools`
   - Public (sichtbar für alle) oder Private (nur für dich)
   - "Add a README file" auswählen
   - "Create repository"

2. **Dateien hochladen:**
   - Lade alle Dateien dieser Ordnerstruktur hoch
   - Oder nutze GitHub Desktop / Git

3. **GitHub Pages aktivieren:**
   - Repository → Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main" → "/ (root)"
   - Save
   - Warte 2-3 Minuten, dann ist die Seite live

## ✏️ Header/Footer ändern

Ändere einfach die Datei `assets/js/header-loader.js` - die Änderungen werden automatisch auf allen Tool-Seiten angezeigt!
